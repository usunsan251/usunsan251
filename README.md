# Hello, World! 🐧

大学院で**音楽情報処理**、**知覚心理学**、**進化計算**の研究をしながら、UnityやWeb技術を用いて「人間の感覚をハックする」プロダクトを開発しています。
技術的な関心は、リアルタイムセンサー解析、物理シミュレーション、そしてユーザー体験の定量化にあります。

- 🌱 **Current Focus:** 音楽同期アルゴリズム / 対話型進化計算(IEC) / 知覚適応のデータ可視化
- 🧠 **Philosophy:** 「当たり前」の感覚を数値化し、エンタメやツールとして再定義する。

## 🚀 Key Projects

### 🎵 Audio & Sensory Interaction
* **RhythmStep - Walking Drum Data Logger**
    * 歩行加速度をリアルタイム解析し、テンポに同期したドラム音を生成するWebアプリ。EMAフィルタによるノイズ除去とWeb Audio APIによる動的音響合成を実装。CSV書き出しによる研究データ収集にも対応。
* **Acronym Decoder**
    * モダンなUI/UXを追求した略語クイズ。グラスモルフィズムを採用した洗練されたデザインと、学習効率を高めるヒントロジックを搭載。

### 🕹 Game Engineering
* **Ludo Project (Unity)**
    * 物理演算（Rigidbody）ベースのダイスシステムと、Nodeベースの経路探索アルゴリズムを組み合わせたデジタルボードゲーム。Cinemachineによる動的なカメラ演出を統合。
* **Tactical Furlong (タクティカル・ハロン)**
    * アナログ競馬ゲームの複雑なルール判定・移動計算を自動化したデジタル管理システム。UXに特化したターン制UIをVanilla JSで構築。

### 🔬 Perception & Science Tools (Unity)
* **IllusionTest2 (V-H Illusion Measurer)**
    * 垂直水平錯覚（V-H錯覚）を定量化する精密測定ツール。被験者の微細な調整プロセス（探索挙動）を時系列データとしてロギング可能。
* **HangedMan_Test (Adaptive Perception Measurer)**
    * 視覚環境の反転（180度回転）に対する脳の適応プロセスを測定。ミリ秒単位の反応速度計測とCSVエクスポート機能を備えた実験基盤。

## 🔬 Research & Exploration
- **Sensory Interaction:** 音楽と歩行の引き込み現象（Entrainment）を利用した歩行支援システムの構築。
- **Human Computing:** 視覚・知覚の特性をUnity上でシミュレートし、人間の適応プロセスを定量化する研究。
- **Interactive Evolution:** 対話型進化計算（IEC）を用いた、ユーザーの感性に寄り添うコンテンツ生成の探求。
